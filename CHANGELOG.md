**1.0.0-beta.0 / 2025-03-02**
- Beta release of Flowgger with following functionalities
  - Basic appenders
    - Basic File Appender
    - Console Appender
    - System Out Appender
    - System Err Appender
  - Dynamic logging using play/pause: support keys, flow names, and log types
  - Custom Layout
  - Custom Appenders
  - Log4jsAppender
  - Filters
    - notFor
    - onlyFor
